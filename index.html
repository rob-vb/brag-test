✦ What's new
  • feat
    Redesigned marketing landing page
    New dark hero, Instrument Serif headline, and a "brag fast = breakfast" lede. Added live image preview, pricing section, and FAQ. Replaced the old placeholder with a proper conversion-optimised layout.
    a3f9c12 @robeverblaze
  • feat
    GitHub App webhook now supports pre-release tags
    The brag.fast cook integration now triggers on pre-release events (alpha, beta, rc) in addition to full releases, so you can generate announcement images earlier in the cycle.
    b71da03 @robeverblaze
  • perf
    Satori render pipeline optimisation — 40% faster generation
    Moved font loading out of the per-request cycle into a warm module-level cache. Combined with lazy Sharp initialisation, cold-start image gen dropped from ~1.4s to ~0.85s on Vercel Edge.
    e22bb8f @robeverblaze
⬡ Bug fixes
  • fix
    Webhook retry loop on Cloudflare R2 upload timeout
    R2 uploads that timed out after 10s were incorrectly returning a 200 to GitHub, causing the webhook to mark success while the image was never stored. Now correctly surfaces a 500 so GitHub retries.
    c90ef44 @robeverblaze
  • fix
    Custom font not rendering in OG images for free-tier users
    Font fallback logic was skipping the user's uploaded font when credit balance was below 5, silently falling back to Inter. Fixed the guard condition — font selection now runs independently of credit checks.
    f44ac1e @robeverblaze
  • fix
    Stripe webhook signature mismatch after plan upgrade
    Users who upgraded mid-billing-period triggered a signature validation error because the webhook secret wasn't being rotated on plan change. Secret is now fetched fresh per event rather than cached on session start.
    9d31b77 @robeverblaze
◈ Maintenance
  • refactor
    Split image generation route into edge + server functions
    Moved the Satori render step to an Edge Function and Sharp post-processing to a Node.js Serverless Function. Cleaner boundary, easier to test independently.
    7f02d5a @robeverblaze
  • chore
    Bump Convex to 1.17, update Better Auth to 0.9.1
    Routine dependency updates. No breaking changes. Convex 1.17 includes improved TypeScript inference for validators.
    2c18e9b @robeverblaze
